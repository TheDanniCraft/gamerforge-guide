Wir freuen uns über jeden Beitrag zu diesem Guide. Egal ob es ein Spielereview, das Verbessern von Rechtschreibfehlern und/oder Tippfehlern, eines Blogartikels oder das Verfassen eines neuen Beitrags im Guide.

Auf dieser Seite findest du alles was du wissen musst.

Wir freuen uns über jeden Beitrag. Daher belohnen wir auch jeden Beitrag den du zu diesem Guide leistest. Dabei giebt es keine festen Rewards für jeden Beitrag. Wir entscheiden das ganze von Fall zu Fall.

# Voraussetzungen

  • Einen kostenlosen Github Account
  • Grundkenntnisse in Markdown(Benötigt wenn du Formatierungen ändern willst)

# Anleitung

  1. Klicke auf Edit this Page unter der Seite die du bearbeiten möchtest. Danach wirst du auf die Github Seite der aktuellen Guideseite weitergeleitet.
  2. Logge dich mit deinem Github Account ein oder registriere dich kostenlos.
  3. Klicke auf Github . Wenn du das Repo nocht nicht geforked hast wird dich Github auf eine Seite weiterleiten auf der du einen Knopf findest. Klicke darauf. Danach musst du vermutlich die Kopie des Repositorys aktuallisieren. Folge dazu einfach Githubs Anweisungen.
  4. Nimm die Änderungen direkt im Guthub Web Editor vor.
  5. Um zu sehen wie deine Änderungen aussehen, klicke auf Preview
  6. Wenn du fertig bist klicke auf
  7. Benenne und Beschreibe deine Änderungen
  8. Um deine Änderungen in den Guide zu übernehmen klicke auf . Hier solltest du möglichst genau Beschreiben was du verändert hast, dies erlaubt es uns besser Nachzuvollziehen was du geändert hast. Wenn du fertig bist klicke nochmals auf .
  9. Jetzt heißt es warten. Deine Änderungen werden vom Team geprüft.+++
